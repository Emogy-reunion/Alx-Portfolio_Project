About Us

I was inspired to create this project based on a personal experience. As someone who spent days searching for the perfect house, I understand the challenges and frustrations of finding the ideal home. This project aims to simplify the process of finding real estate listings, making it easier for individuals to discover their dream homes.

This project is a Portfolio Project for Holberton School. You can find the code repository on GitHub.
